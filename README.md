Описание
Данное приложение предназначено для сравнения двух полиномов от нескольких переменных, вводимых в стандартном виде. Переменные указываются в виде: x, x1, …,xn

Требования
* Python версии не ниже 3.4

Состав
* Сравнение многочленов и консольная версия программы: polinomial.py
* Перевод в обратную польскую нотацию: rpn.py
* Тесты: .idea/Tests.py

Консольная версия
Справка по запуску: ./polinomial.py --help
Пример запуска: ./polinomial.py –- “x” “x^2”

Подробности реализации
Модули, отвечающие за перевод в обратную польскую нотацию и сравнение полиномов в точках, расположены в корневой папке. Функции get_numbers() возвращает список чисел и переменных в введённом полиноме, функция sort_variables() оставляет переменные в полученном списке, в методе compare() ведётся подстановка случайного целого числа в диапазоне от -100 до 100 и происходит сравнение полиномов в точке. Перевод в обратную польскую нотацию осуществляется с помощью функций unary_minus_handler(), add_multiplication(), polynomial_to_list(), преобразующие введённый полином в список с учётом недостающих знаков умножения и унарных минусов, и с помощью функций operators_handler(), extract_left_operators() и interpret() происходит сам перевод в обратную польскую нотацию. Функция compute() вычисляет значение выражения.
